# CLI File Management Challenge

## Overview
This project demonstrates basic file and directory management using command-line tools in Git Bash (MINGW64).  
The exercise focuses on practicing essential shell commands such as creating files, moving files, copying files, renaming files, and deleting directories.

## Objective
- Practice fundamental command-line operations
- Understand file and directory manipulation
- Gain familiarity with common Unix-style commands

## Commands Used

## 1. Navigate to the Documents Directory
- cd documents

## 2. Create a New Folder
- mkdir cli_challenge

## 3. Create Three Files
- touch file1.txt file2.txt file3.txt

## 4. Create a Backup Folder
- mkdir backup

## 5. Move a File to the Backup Folder
- mv file1.txt backup

## 6. Rename a File
- mv file2.txt renamed_file.txt

## 7. Copy the Renamed File to the Backup Folder
- cp renamed_file.txt backup

## 8. Delete a File
- rm file3.txt

## 9. Remove the Backup Directory and Its Contents
- rm -r backup

## 10. Return to the Previous Directory
- cd ..

## 11. Delete the Challenge Folder
- rm -r cli_challenge

## Key Concepts Practiced
- Directory navigation
- File creation
- File movement and renaming
- File copying
- File and directory deletion
